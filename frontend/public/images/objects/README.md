## Как заменить заглушки на реальные изображения

1. Поместите реальные фотографии объектов в эту папку с правильными именами (см. список выше)
2. Файлы автоматически заменят существующие заглушки
3. Система автоматически найдет изображения по названию объекта

**Примечание:** Все изображения-заглушки уже созданы и система работает. Вы можете заменить их на реальные фотографии в любое время.

## Примечания

- Все изображения должны быть оптимизированы для веб-использования
- Рекомендуемый формат: JPG с качеством 80-85%
- Размер файла: не более 500KB на изображение
- Соотношение сторон: 4:3 или 16:9
- Если изображение не найдено, будет использован `default.jpg`
